# Counter App

A simple mobile application that demonstrates basic UI interaction by incrementing a counter using a single button.

## Overview
This application contains one screen that displays a numeric counter and a button. Each time the button is pressed, the counter value increases by one. The app focuses on core mobile development fundamentals without adding unnecessary complexity.

## Features
- Single screen user interface  
- Counter displayed on the screen  
- Button to increment the counter  
- Immediate UI update on button press  
- Clean and readable code  

## Out of Scope
The following features are intentionally not included:
- Backend integration  
- User authentication  
- Data persistence or local storage  
- Database usage  
- Advanced state management  

## Tech Stack
- Framework: Flutter  
- Language: Dart  
- State Management: Basic StatefulWidget
